# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

This is a curated list of restaurants and activities in the Indianapolis area that the group has visited or wants to visit. It's a static markdown document organized by location (Broad Ripple, Downtown, Castleton, Fishers, etc.) with checkmarks indicating visited vs. unvisited establishments.

## Structure

- **README.md** — The main content file containing all restaurant and activity listings
  - Each location has its own section (## Headers)
  - Restaurants use checkboxes: `[x]` for visited, `[ ]` for not yet visited
  - Entries include: restaurant name (linked to Google Maps), address, and a brief description

## Editing Guidelines

- **Adding a new restaurant:** Include a checkbox (checked or unchecked), restaurant name with Google Maps link, address, and description with cuisine/atmosphere details
- **Format:** Follow the existing pattern with markdown links and consistent indentation
- **Locations:** Add new regions as `##` headers as needed; keep them in a logical geographic order
